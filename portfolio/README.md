# Developer Portfolio

Welcome to my personal developer portfolio! This project showcases my skills, projects, and professional background. It is built using modern web development technologies to ensure a fast, responsive, and visually appealing user experience.

## ✨ Features

- **Modern Tech Stack**: Built with React and Vite for blazing-fast performance.
- **Responsive Design**: Fully responsive layout that looks great on desktops, tablets, and mobile devices.
- **Project Showcase**: Detailed sections for highlighted projects, including descriptions, technologies used, and links to live demos or source code.
- **Vercel Ready**: Pre-configured for seamless deployment to Vercel (`vercel.json` included).

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm (or yarn/pnpm) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   # or yarn install / pnpm install
   ```

### Running Locally

Start the development server:

```bash
npm run dev
# or yarn dev / pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## 🛠️ Built With

- [React](https://reactjs.org/) - A JavaScript library for building user interfaces
- [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling
- Frontend Routing (configured for Single Page Applications)

## 📦 Deployment

This project is configured to be easily deployed on Vercel. 

1. Push your code to a GitHub repository.
2. Import the project in Vercel.
3. The included `.vercel` ignores and `vercel.json` routing configuration will automatically be detected and applied.
